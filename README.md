## Description

This Express.js app uses MySQL to create and connect an e-commerce back-end API. 

## Installation

1. Clone the repo
```sh
git clone https://github.com/your_username_/Project-Name.git
```
2. Install NPM packages
```sh
npm install
```
3. NPM run seed, then nodemon server
```sh
nodemon ./server.js
```
